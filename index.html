My Cheatsheet for HTML and CSS

HTML

This section is to point out the different html tags and how to use them. It will be used as my own personal reference to look up quickly how to write html without going to the internet.

Text Tags

Tag Name Description
<!doctype html> document type used at the first line of code to denote what type of document the file is, usually HTML
<html> HMTL denotes that the following code is going to be HTML
<title> Title used to give the website a title displayed in the tab
<head> Head used to mark the head of the page, links to any CSS docs and any other links pertaining to the backend go here
<h1-h6> headings 1-6 Headings or titles for the website, used to start different sections within the website. h1 being the largest and h6 being the smallest
<p> paragraph this is used to write out any content onto the actual website
<link> Link used to link other websites or docs to the website. when using to link css, use relative path and relationship to html
<img> Image used to add images to website. If image is in same directory, use relative path based on where the HTML doc is, otherwise use full URL of image
<div> divide used to make a space between elements. useful to make sections easier to read
